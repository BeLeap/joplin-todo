# 남은 작업 정리 (기준: implementation-plan + phase 구현 문서)

이 문서는 현재 구현 상태를 기준으로, MVP 완료 및 배포 준비를 위해 남은 작업을 단계별로 정리한 체크리스트입니다.

## 1) 우선순위 높음 (MVP 완료 필수)

- [ ] **실제 OneDrive 연동 구현**
  - [ ] OAuth 로그인 플로우 연결 (시스템/내장 브라우저)
  - [ ] OneDrive 내 Joplin 동기화 파일 목록 조회
  - [ ] Mock 소스(`MockOneDriveJoplinSource`)를 실소스 구현체로 교체
  - [ ] 비암호화 실계정에서 TODO 로딩 성공 확인

- [x] **Joplin 실데이터 파싱 고도화**
  - [ ] 샘플/실계정 데이터 기준 파싱 안정성 보강
  - [x] 에지 케이스(빈 제목, malformed 필드, 누락된 시간값) 처리
  - [x] E2EE 감지 시 미지원 안내 UX를 앱 화면과 일관되게 노출

- [ ] **Android 홈 화면 위젯 네이티브 구현 완성**
  - [ ] `AppWidgetProvider` 구현
  - [ ] RemoteViews 리스트 레이아웃 바인딩
  - [ ] 수동 새로고침 액션 연결
  - [ ] 주기 갱신 Worker 연결
  - [ ] 앱 캐시 ↔ 위젯 저장소 브리지(`WidgetBridge`)를 네이티브 구현체로 교체

## 5) 제안 실행 순서

1. 실제 OneDrive OAuth + 파일 조회 연결
2. 실데이터 파서 안정화 및 E2EE UX 정리
3. Android 위젯 네이티브(AppWidgetProvider/Worker/Bridge) 구현
4. 통합 검증(실계정 + 오프라인 + 주기갱신)
5. 배포 준비 및 내부 테스트 빌드 검수
